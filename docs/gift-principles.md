# 送礼推荐原则 v0.2

这份文档用于指导礼物推荐数据生成和推荐排序。目标不是写泛泛的送礼攻略，而是把“什么东西更适合作为礼物”沉淀成可执行规则。

## 产品边界

本产品主要推荐电商、本地电商、O2O 能覆盖的礼物方向。

- 主体应是普通用户可以搜索、下单、配送、到店购买或预约核销的商品/服务。
- 高频推荐应优先选择已成品、中小件、好包装、好配送、低专业门槛的品类。
- 体验、课程、票券、定制服务可以出现，但默认是补充层，不应成为主体。
- 不推荐烟酒、投资理财、医疗功效、保健品疗效、奢侈品炫耀。
- 不把大件家具、装修设备、维修替换件、米面油补货、清洁耗材放到默认主推荐位。

一句话原则：不是挑“最有用”的东西，而是挑“对人、对关系、对场景都成立，并且能顺利成交履约”的东西。

## 四层判断

推荐一个礼物前，先判断四件事。

### 1. 礼物性

它是否像礼物，而不只是日用品、采购品或补货品。

高礼物性信号：

- 有印象点：能被记住、可展示、可佩戴、可开箱、可长期触达。
- 有关系表达：能说明“我知道你喜欢什么”或“我认真想过你”。
- 有包装感：礼盒、套装、门店包装、卡片、礼袋、适合郑重交付。
- 具体可买：用户能搜索到商品方向，而不是只看到“化妆品”“数码产品”。

降权信号：

- 谁都能用的补货品。
- 维修件、替换件、耗材、清洁用品。
- 大件、占地方、需要安装维护、退换麻烦。
- 看起来像公司行政采购或家庭采购。

### 2. 可履约性

它是否能被电商、本地电商或 O2O 顺利覆盖。

优先：

- 电商供给密集：美妆香水、包袋配饰、数码配件、桌面办公、咖啡茶饮、食品甜品、运动户外、家居小家电、书影音、IP 周边。
- O2O 履约成熟：鲜花蛋糕、写真、调香、烘焙、手作、展览、运动体验、同城甜品水果礼盒。
- 可包装、可配送、可退换、可预约。

谨慎：

- 地点和时间强约束的服务。
- 需要对方配合的体验。
- 制作周期长、不可退换的定制物。
- 难以搜索到稳定供给的过窄商品。

### 3. 风险可控性

推荐系统不能只负责“看起来浪漫”，还要尽量避免翻车。

高风险点：

- 肤质、色号、气味、尺码、材质过敏。
- 设备接口、型号、系统兼容。
- 专业参数、圈层知识、已有装备冲突。
- 食品忌口、过敏、保质期、冷链。
- O2O 的地点、时间、预约、退款。
- 定制礼物的审美、周期、不可退换。

风险不是必须排除，但要进入 `riskTags` 或推荐理由。

### 中国式渠道和合规底线

中国语境下，父母礼物、节日拜访和女性亲密关系礼物，经常会涉及保健食品、营养品、护肤美妆和滋补礼盒。这些品类可以推荐，但要比普通商品更重视渠道和表达方式。

- 保健食品、营养品可以作为父母礼物的重要候选，但只能表达健康关怀，不能宣称治疗、预防疾病或替代药物。
- 推荐保健食品时，优先正规渠道和可核验信息，例如官方旗舰店、连锁药房、大型商超、品牌自营店；风险提示可写“认准标识”“看适宜人群”“按说明食用”。
- 普通食品不要写保健功效。滋补礼盒、低糖食品、蛋白粉、维生素、钙片等可以推荐，但要避免“降血糖”“治失眠”“改善三高”等医疗化表达。
- 化妆品、护肤品、香水可以作为妈妈、恋人、闺蜜的重要礼物，但优先专柜、官方旗舰店、品牌自营和可退换渠道；不要推荐来路不明代购。
- 对中老年父母，礼物文案要避免“你老了/你病了/你需要修复问题”的暗示，改成“体面升级、日常关心、舍不得买的好东西”。

### 4. 上下文匹配

同一个商品，在不同关系、节日、季节、人格倾向下礼物感不同。

推荐时应考虑：

- 送给谁：恋人、闺蜜、父母。
- 什么场合：生日、纪念日、节日、道歉/和好、日常关心。
- 具体节日：520、七夕、情人节、母亲节、父亲节、毕业季、圣诞、新年、中秋等。
- 当前季节和天气：不要在明显不合季的时间高频推荐围巾、暖手宝、厚毯；夏季优先清爽、出行、防晒、香氛、冷饮、轻户外等。
- TA 的 E/I 倾向：E 人可提高社交、展示、共同体验权重；I 人可提高独处、居家、桌面、阅读、低社交负担权重。
- 送礼人是否懂对方的专业兴趣：不懂时不要硬送专业装备。

## 关系边界

### 恋人

适合更高的仪式感、记忆点和日常可见触点。

适合：

- 香水、饰品、包袋、情侣体验、生日蛋糕、花束搭配、影像记录、对方长期想要的物件。
- 纪念日、520、七夕更适合有象征意义和包装感的礼物。

慎重：

- 暗示身体问题、卫生问题、外貌问题的礼物。
- 过于实用的大件，例如椅子、大家电、收纳柜。
- 关系阶段不匹配的贵重礼物。

### 闺蜜/好友

要区分普通朋友和好闺蜜。在中国语境里，好闺蜜往往非常亲密，可以送更贴近变美、悦己、身体护理、共同体验的东西；普通朋友则要保守一些。

适合：

- 美妆香水、护肤身体护理、包袋配饰、睡衣家居服、手帐文具、追星周边整理、写真、咖啡甜品、展览体验。
- 好闺蜜之间可以推荐更“姐妹感”的礼物，例如专柜彩妆、香水试香、身体乳礼盒、美容仪入门款、真丝眼罩、睡衣礼盒、首饰、拍照写真。
- 推荐理由可以强调“懂她审美”“一起变美”“陪她放松”“适合姐妹约会”，不要写成恋人式暧昧表达。

慎重：

- 普通朋友关系里太私密、太贵重、太像伴侣专属的东西。
- 强烈依赖尺码、肤质、身体特征的礼物；好闺蜜也要在已知偏好时才推荐。
- 内衣、塑形、祛痘、脱毛、减肥、体味相关方向容易冒犯，除非用户明确提出。

### 父母

父母礼物在中国语境里不应只理解成“实用升级”。很多父母，尤其 40-60 岁妈妈，既在意健康，也在意体面和变美，只是平时不舍得为自己花钱。推荐要兼顾健康关怀、体面爱美、实用升级和陪伴表达。

适合：

- 健康关怀但不宣称疗效：正规保健食品、营养补充品、蛋白粉/钙片/维生素类礼盒、低糖食品礼盒、体检套餐提醒类服务。
- 妈妈爱美和体面：抗老护肤礼盒、身体护理礼盒、口红/粉底/气垫等彩妆、香水小样、丝巾、珍珠/银饰、护发护理。
- 爸妈通用的舒适升级：阅读灯、智能屏/音箱、茶咖器具、旅行用品、保暖配饰、小家电。
- 节日拜访和家庭场景：水果礼盒、糕点礼盒、茶礼盒、地方特产礼盒、鲜花蛋糕搭配。

慎重：

- 暗示衰老、疾病、矫正、治疗的礼物。保健食品和营养品只能表达关心，不能写治疗、预防疾病或替代药物。
- 来路不明的保健品、进口营养品、代购护肤品。优先官方旗舰店、连锁药房、大型商超、品牌专柜或可核验渠道。
- 复杂数码、学习成本高的设备。
- 需要长期维护或占地方的大件。

## 具体节日策略

### 520 / 七夕 / 情人节

关键词：浪漫、象征、仪式、可拍照、可开箱。

适合：

- 鲜花蛋糕搭配、美妆香水、饰品、包袋、情侣体验、写真、香氛、纪念但不过度定制的物件。

慎重：

- 过度实用的大件、低价值小物、像临时采购的东西。

### 生日

关键词：个人偏好、被看见、愿望清单。

适合：

- 对方明确喜欢或长期想要的东西。
- 有个性但低风险的品类：香水小样、通勤包、桌面设备、IP 周边、精品咖啡、影像记录。

### 纪念日

关键词：共同经历、关系确认、长期保存。

适合：

- 首饰、香水、共同体验、票根收藏、旅行相关、纪念摆件。

慎重：

- 相册/照片书可以少量出现，但不要成为默认答案。

### 母亲节 / 父亲节

关键词：体面、健康关怀、爱美、舒适升级、陪伴、不暗示问题。

适合：

- 花束蛋糕搭配、护肤/身体护理礼盒、正规营养补充品、茶咖器具、阅读灯、旅行用品、舒适家居小物、易用小家电。
- 给妈妈可以提高护肤、美妆、丝巾、首饰、香氛身体护理的权重；给爸爸可以提高茶咖、通勤配件、轻运动、剃须护理、易用数码的权重。

慎重：

- 强医疗暗示、保健品疗效、复杂设备、来路不明代购。

### 毕业季 / 入学 / 入职

关键词：新阶段、通勤、学习、桌面、祝福。

适合：

- 通勤包、品质钢笔、桌面灯、机械键盘成品、降噪耳机、证件包、旅行箱配件。

### 中秋 / 新年 / 春节

关键词：家庭、团圆、拜访、体面包装。

适合：

- 食品礼盒、茶咖礼盒、水果礼盒、家居小家电、父母友好型升级。

慎重：

- 过于私人、难分享、包装不体面的东西。

## 季节策略

季节不应硬性决定礼物，但能明显影响推荐优先级。

- 春季：花束、香氛、轻户外、野餐、旅行、护肤修护、拍照。
- 夏季：防晒周边、清爽香氛、冷萃咖啡、轻运动、露营灯、旅行收纳、清凉小家电。
- 秋季：咖啡茶饮、通勤配饰、阅读、桌面升级、城市户外、影像记录。
- 冬季：围巾手套、保暖家居、热饮器具、暖光灯、身体护理、节日礼盒。

明显反季节商品要降权。例如夏季不应高频推荐厚围巾、暖手宝、厚毛毯；冬季不应高频推荐清凉喷雾、冰饮器具，除非有明确场景。

## E/I 倾向

E/I 不是人格标签结论，只是软权重。

### 偏 E

提高这些方向权重：

- 可展示、可社交、可一起参与、可拍照传播。
- 例如演唱会补给、写真、调香/烘焙体验、通勤包、饰品、桌游、派对甜品、运动体验。

### 偏 I

提高这些方向权重：

- 独处舒适、低社交负担、桌面/居家/阅读/创作。
- 例如阅读灯、桌面灯、香氛、降噪耳机、咖啡茶饮、书影音、手帐文具、居家小家电。

不要把 E/I 当成绝对规则。用户明确偏好优先。

## 专业品类原则

不要送对方专业领域里的“核心装备”，除非送礼人明确知道参数、型号或愿望清单。

容易翻车的专业品类：

- 鱼竿、线轮、拟饵、专业钓具。
- 相机镜头、机身附件、专业摄影器材。
- 骑行零件、轮组、刹车、锁踏。
- 机械键盘高端套件、键帽、轴体。
- 专业耳机、桌面音频、麦克风。
- 滑雪板、雪靴、绑定。
- 乐器、效果器、音箱。
- 模型喷涂、改造工具、专业手作工具。

判断规则：

- 如果需要查兼容表、型号树、接口、尺码体系、调性、阻抗、绑定、配列，就默认降权。
- 如果对方给过链接、愿望清单、精确型号，可以推荐。
- 如果送礼人不懂，优先推荐周边、收纳、展示、耗材礼盒、体验券，而不是核心装备。

例如：不懂钓鱼时，不要默认推荐鱼竿；可以推荐防晒服、户外水壶、渔具收纳包，但仍要提示“需确认钓法/装备习惯”。

## 高频推荐品类

这些品类适合成为默认主力。

| 一级品类 | 典型方向 | 适合人群 | 风险提示 |
|---|---|---|---|
| 美妆护肤与身体护理 | 口红礼盒、护手霜礼盒、身体乳套装、化妆刷套装 | 恋人、闺蜜、母亲节 | 看色号、看肤质、正品渠道 |
| 香水与家居香氛 | 试香套组、旅行装、无火香薰、香薰蜡烛 | 恋人、闺蜜、居家人群 | 看气味、可能晕香 |
| 父母营养与滋补 | 正规营养品、蛋白粉、钙片、维生素、低糖食品、滋补礼盒 | 父母、长辈 | 不写疗效、看适宜人群、正规渠道 |
| 首饰包袋与配饰 | 项链、手链、卡包、托特包、围巾、证件包 | 恋人、闺蜜、上班族 | 看风格、看材质 |
| 数码配件 | 降噪耳机、成品键盘、桌面充电站、移动电源 | 学生、上班族、数码爱好者 | 看兼容、看已有设备 |
| 桌面办公 | 品质钢笔、桌面灯、支架、设计感笔记本 | 学生、上班族、创意人群 | 避免普通文具感 |
| 食品甜品 | 蛋糕、巧克力、曲奇、水果礼盒、低糖点心 | 广泛适用 | 看忌口、看保质期 |
| 咖啡茶饮 | 挂耳咖啡、精品豆、冷萃壶、茶叶品鉴礼盒 | 上班族、父母、朋友 | 看口味、看使用门槛 |
| 运动户外 | 训练包、运动水壶、跑步腰包、露营灯、野餐垫 | 学生、上班族、轻户外人群 | 看运动习惯 |
| 居家小家电 | 胶囊咖啡机、早餐机、阅读灯、香薰机 | 父母、恋人、自住人群 | 看空间、看维护 |
| 旅行通勤 | 通勤包、证件包、旅行收纳、颈枕升级款 | 上班族、学生、旅行人群 | 看尺寸、看出行频率 |
| 追星/IP/游戏动漫 | 卡册、票根册、展示架、手办展示盒、手柄 | 兴趣明确人群 | 看 IP/版本/已有收藏 |
| 本地生活 O2O | 写真、调香、烘焙、展览、运动体验 | 恋人、闺蜜、朋友 | 需预约、看地点 |

## 谨慎推荐品类

这些品类可以极少数出现，但不应默认高频。

| 品类 | 为什么容易翻车 | 可以推荐的条件 | 风险提示 |
|---|---|---|---|
| 专业钓具 | 钓法、水域、目标鱼、调性都影响选择 | 对方给过清单或链接 | 需确认钓法 |
| 相机镜头 | 卡口、焦段、用途强相关 | 已知机身和想要焦段 | 需确认型号 |
| 骑行零件 | 兼容复杂 | 已知整车配置 | 需懂参数 |
| 客制化键盘套件 | 配列、轴体、键帽高度都主观 | 对方明确指定 | 需确认配列 |
| 专业音频 | 阻抗、接口、使用场景复杂 | 已知设备链路 | 需确认接口 |
| 乐器/效果器 | 手感和音色高度主观 | 对方指定型号 | 需试用 |
| 大件家具 | 占空间、安装、共同决策 | 对方明确需要 | 需确认尺寸 |
| 强定制礼物 | 审美错就不可退 | 风格确定、周期充足 | 不易退换 |

## 近年新兴方向

可以作为生成候选时的灵感池，但仍要服从电商/O2O 可履约和低翻车原则。

- AI 小硬件和智能穿戴：AI 眼镜、智能戒指、智能手表周边、翻译/记录类小设备。不要写医疗疗效。
- 桌面生产力：成品键盘、桌面灯、充电底座、扩展坞、支架、电子纸小屏。
- 香氛和身体护理：试香、身体乳、护手霜、家居香氛、放松类护理。
- 精品咖啡、茶饮、无酒精社交饮品：挂耳、精品豆、冷萃、无酒精起泡饮。
- 宠物友好生活方式：宠物肖像、宠物收纳、智能清洁、宠物友好家居。不要默认送宠物食品。
- 城市户外：露营灯、野餐垫、轻量背包、运动腰包、骑行通勤配件。
- 影像记录：拍立得、相纸、照片打印、票根收藏、复古数码。
- 追星和票根经济：卡册、票根册、展示架、演唱会补给。
- 游戏动漫和潮玩：正版周边、展示收纳、手柄、桌游、卡牌收纳。
- 本地生活体验：写真、调香、烘焙、手作、展览、运动体验。

## 可直接进入 Prompt 的规则

- 主体只推荐电商/O2O 可履约的礼物方向。
- 每次生成应明确品类，不要自由发散。
- 商品方向必须具体到可搜索购买。
- 优先已成品、中小件、好包装、好配送、低专业门槛。
- 推荐理由要说明为什么它适合作为礼物，而不是只说明功能。
- 预算由商品方向自然决定，不要为了便宜生成低价值感小物。
- 高风险品类必须给出简短风险提示。
- 中国语境下，父母礼物要覆盖健康关怀和体面爱美；闺蜜礼物要区分普通朋友和好闺蜜，好闺蜜可以更亲密但仍要看偏好。
- 对专业品类，不懂不乱送；没有清单、型号、链接时不要推荐核心装备。
- 结合具体节日调整礼物感：520/七夕偏浪漫仪式，生日偏个人偏好，父母节日偏舒适升级。
- 结合季节调整优先级，避免明显反季节商品。
- 结合 E/I 倾向做软权重，但用户明确偏好优先。
- 不要使用性别刻板印象、奢侈炫耀、医疗疗效、保健品疗效、烟酒、投资理财。
